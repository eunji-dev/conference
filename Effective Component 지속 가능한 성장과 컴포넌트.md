## 회고

```
컴포넌트를 props로 넘기는 방식을 좀 더 잘 활용하면 좋을거같다.

컴포넌트를 props로 넘기는 방식을 사용해서 리팩토링을 해봤는데 props drilling이 줄어 들었고 조건에 따라 컴포넌트를 보여줄때도 코드가 좀 더 깔끔해진것 같다.

그리고 변경에 좀 더 유연한 컴포넌트를 만들 수 있게 된 것 같다.
```

> Headless 기반 추상화 <br>

1. 데이터 관리

3. 사용자와의 상호작용
